### 開発セットアップ

パッケージマネージャは[bun](https://bun.sh/)を採用

#### Install Bun

```bash
curl -fsSL https://bun.sh/install | bash
```

#### Install node_modules

```bash
bun install
```

#### Compile Sass

```bash
bun compile:sass
```
